bevel

the first keyboard I've designed from the ground up, and has been the longest I've consistently used a single keyboard since getting into this mess. Currently on the second revision, the first try had a few issues - nothing that couldn't be worked around, but just annoying enough that I wanted to fix them. I use a slightly-modified footprint for the pro-micro, so the circuit board is flippable for each half but the microcontroller is face-up on both halves.

designing the bevel was a lot of fun, since it meant an opportunity to learn pcb design, which was much less scary than expected. I use a program called Kicad, which is free and open-source.
